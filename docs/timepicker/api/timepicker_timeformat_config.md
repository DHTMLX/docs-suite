# timeFormat

@short: Optional. Defines what clock format is activated: the 12-hour or 24-hour one

@signature: {'timeFormat?: 12 | 24;'}

@default: 24

@example:
const timepicker = new dhx.Timepicker("timepicker_container", {
	timeFormat: 12
});

@descr:
