# resizable

@short: Optional. Defines whether columns can be resized

@signature: {'resizable?: boolean;'}

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
    columns: [
        // columns config
    ],
    data: dataset,
    resizable: true
});
