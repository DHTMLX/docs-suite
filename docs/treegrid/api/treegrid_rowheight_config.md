# rowHeight

@short: Optional. Defines the height of a row in a grid

@signature: {'rowHeight?: number;'}

@default: 40

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
	columns: [
		// columns config
	],
	rowHeight: 30,
	data: dataset
});
