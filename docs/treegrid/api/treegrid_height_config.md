# height

@short: Optional. Sets the height of a grid or adjusts it automatically to the content

@signature: {'height?: number | "auto";'}

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
	columns: [
		// columns config
	],
	height: 400,
	data: dataset
});

@descr:
