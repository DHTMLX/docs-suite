# width

@short: Optional. Sets the width of a grid

@signature: {'width?: number;'}

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
    columns: [
        // columns config
    ],
    width: 400,
    data: dataset
});

@descr:
