# collapsed

@short: Optional. Defines that a treegrid is initialized in the collapsed state

@signature: {'collapsed?: boolean;'}

@default: false

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
    columns: [
		// columns config
	],
    data: dataset,
    collapsed: true
});
