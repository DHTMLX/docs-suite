# dragCopy

@short: Optional. Defines that a row is copied to a target during drag-n-drop

@signature: {'dragCopy?: boolean;'}

@example:
const source = new dhx.TreeGrid("treegrid-source", {
    dragMode: "source", 
    dragCopy: true
});

const target = new dhx.TreeGrid("treegrid-target", {
    dragMode: "target", 
    dragCopy: true
});
