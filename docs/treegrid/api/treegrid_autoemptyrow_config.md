# autoEmptyRow

@short: Optional. Adds an empty row after the last filled row in the TreeGrid

@signature: {'autoEmptyRow?: boolean;'}

@default: true

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
	columns: [
		// columns config
	],
	autoEmptyRow:true,
	data: dataset
});
