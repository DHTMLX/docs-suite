# scrollTo()

@short: scrolls a grid to a specified cell

@signature: {'scrollTo(rowId: string | number, colId: string | number): void;'}

@params:
- `rowId: string | number` - the id of a row
- `colId: string | number` - the id of a column

@example:
grid.scrollTo("15","c");

@descr:
