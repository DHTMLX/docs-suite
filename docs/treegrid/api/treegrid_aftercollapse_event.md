# afterCollapse

@short: fires after collapsing a treegrid

@signature: {'afterCollapse: (rowId: string | number) => void;'}

@params:
- `rowId: string | number` - the id of a collapsed row

@example:
treeGrid.events.on("afterCollapse", function(rowId) {
    // your logic here
});
