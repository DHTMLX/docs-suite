# bottomSplit

@short: Optional. Sets the number of frozen rows from the bottom

@signature: {'bottomSplit?: number;'}

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
	columns: [
		// columns config
	],
	bottomSplit: 2,
	data: dataset
});
