# rightSplit

@short: Optional. Sets the number of frozen columns from the right

@signature: {'rightSplit?: number;'}

@example:
const treegrid = new dhx.TreeGrid("treegrid_container", {
	columns: [
		// columns config
	],
	rightSplit: 2,
	data: dataset
});
