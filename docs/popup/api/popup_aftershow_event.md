# afterShow

@short: fires after a popup is shown

@signature: {'afterShow: (node: HTMLElement) => void;'}

@params:
- `node: HTMLElement` - the container a popup is placed in

@example:
popup.events.on("afterShow", function(element){
    console.log("A popup is shown");
});

@descr:
