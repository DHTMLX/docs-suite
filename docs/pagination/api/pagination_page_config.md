# page

@short: Optional. The index of the initial page set in the pagination

@signature: {'page?: number;'}

@default: 0

@example:
const pagination = new dhx.Pagination("pagination_container", {
    css: "dhx_widget--bordered",
    page: 4 
});

@descr:
