# mode

@short: Optional. Specifies the mode of displaying a tabbar

@signature: {'mode?: "left" | "right" | "bottom" | "top";'}

@default: "top"

@example:
const tabbar = new dhx.Tabbar("tabbar_container", {
    mode: "bottom"
});

@descr:
