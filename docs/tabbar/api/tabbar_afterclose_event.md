# afterClose

@short: fires after closing a tab in Tabbar

@signature: {'afterClose: (id: string) => void;'}

@params:
- `id: string` - the id of a closed tab

@example:
tabbar.events.on("afterClose", function(id){
    console.log(id);
});

@descr:
