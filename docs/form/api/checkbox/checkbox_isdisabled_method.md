# isDisabled()

@short: checks whether a Checkbox control is disabled

@signature: {'isDisabled(): boolean;'}

@returns:
`true`, if a control is disabled. Otherwise, `false`.

@example:
form.getItem("checkbox").isDisabled(); // -> true/false
