# isVisible()

@short: checks whether a Toggle control is visible on the page

@signature: {'isVisible(): boolean;'}

@returns:
`true`, if a control is visible. Otherwise, `false`.

@example:
form.getItem("toggle").isVisible(); // -> true/false
