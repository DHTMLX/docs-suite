# isSelected()

@short: checks whether the selected state is enabled

@signature: {'isSelected(): boolean;'}

@returns:
`true`, if the control is in the selected (pressed) state. Otherwise, `false`.

@example:
form.getItem("toggle").isSelected(); // -> true/false
