# getValue()

@short: returns the current value of a ColorPicker control (in the Hex format)

@signature: {'getValue(): string;'}

@returns:
A string with the current value of the control.

@example:
const value = form.getItem("colorpicker").getValue();
// -> "#90D2AF"
