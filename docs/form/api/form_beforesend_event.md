# beforeSend

@short: fires before a form is sent to the server

@signature: {'beforeSend: () => boolean | void;'}

@returns:
Return `true/false` to allow/forbid a form sending to the server.

@example:
form.events.on("beforeSend", function(){
   // your logic here
   return true;
});

@descr:
