# isVisible()

@short: checks whether a DatePicker control is visible on the page

@signature: {'isVisible(): boolean;'}

@returns:
`true`, if a control is visible; otherwise, `false`.

@example:
form.getItem("datepicker").isVisible(); 
// -> true/false
