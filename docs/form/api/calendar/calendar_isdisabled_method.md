# isDisabled()

@short: checks whether a DatePicker control is disabled

@signature: {'isDisabled(): boolean;'}

@returns:
`true`, if a control is disabled; otherwise, `false`.

@example:
form.getItem("datepicker").isDisabled(); 
// -> true/false
