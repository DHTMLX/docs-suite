# cols

@short: Optional. Arranges controls inside the control group horizontally

@signature: {'cols?: object[];'}

@example:
const form = new dhx.Form("form_container", {
    css: "dhx_widget--bordered",
    width: "250px",
    rows: [
        {
            padding: "10px",
            cols: [{
                type: "checkbox",
                width: "100px",
                label: "I agree",
                name: "agree",
                id: "agree",
                checked: true
            },
            {
                type: "checkbox",
                label: "I disagree",
                name: "align",
            }]
        }
    ]
});

@descr:
