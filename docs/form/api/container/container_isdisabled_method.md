# isDisabled()

@short: checks whether a Container control is disabled

@signature: isDisabled(): boolean;

@returns:
`true`, if a control is disabled, otherwise `false`.

@example: form.getItem("container").isDisabled();
// -> true/false

@changelog: added in v7.2
