# isVisible()

@short: checks whether a Container control is visible on the page

@signature: isVisible(): boolean;

@returns:
`true`, if a control is visible, otherwise `false`.

@example: form.getItem("container").isVisible();
// -> true/false

@changelog: added in v7.2
