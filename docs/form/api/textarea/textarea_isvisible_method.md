# isVisible()

@short: checks whether a Textarea control is visible on the page

@signature: {'isVisible(): boolean;'}

@returns:
Return `true`, if a control is visible; otherwise, `false`.

@example:
form.getItem("textarea").isVisible(); 
// -> true/false
