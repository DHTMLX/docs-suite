# isDisabled()

@short: checks whether a Textarea control is disabled

@signature: {'isDisabled(): boolean;'}

@returns:
Return `true`, if a control is disabled; otherwise, `false`.

@example:
form.getItem("textarea").isDisabled(); 
// -> true/false
