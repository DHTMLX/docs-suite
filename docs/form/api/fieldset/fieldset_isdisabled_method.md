# isDisabled()

@short: checks whether a Fieldset control is disabled

@signature: {'isDisabled(): boolean;'}

@returns:
`true`, if the control is disabled; otherwise, `false`.

@example:
form.getItem("fieldset").isDisabled(); 
// -> true/false
