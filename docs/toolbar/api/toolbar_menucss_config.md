# menuCss

@short: Optional. Adds style classes to all containers of Toolbar controls with nested items 

@signature: {'menuCss?: string;'}

@example:
<style>
    .my_first_class {
        /*some styles*/
    }
 
    .my_second_class {
        /*some styles*/
    }
</style>

const toolbar = new dhx.Toolbar("toolbar_container", {
    menuCss:"my_first_class my_second_class"
});
