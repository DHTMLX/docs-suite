# unselect()

@short: unselects a selected Toolbar item

@signature: {'unselect(id?: string | number): void;'}

@params:
- `id: string | number` - optional, the id of a Toolbar item

@example:
// unselects all previously selected items
toolbar.unselect();
 
// unselects a specified selected item
toolbar.unselect("web");

@descr:

**Related sample**: [Toolbar. Select / unselect items](https://snippet.dhtmlx.com/mi7qjwg2)
