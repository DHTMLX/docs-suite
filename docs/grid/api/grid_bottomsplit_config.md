# bottomSplit

@short: Optional. Sets the number of frozen rows from the bottom

@signature: {'bottomSplit?: number;'}

@example:
const grid = new dhx.Grid("grid_container", {
	columns: [
		// columns config
	],
	bottomSplit: 2,
	data: dataset
});
