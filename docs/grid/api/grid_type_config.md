# type

@short: Optional. Sets the TreeGrid mode of a grid

@signature: {'type?: "tree";'}

@example:
const grid = new dhx.Grid("grid_container", {
	type: "tree",
    columns: [
        // columns config
    ],
    width: 400, 
    data: dataset
});

@descr:
