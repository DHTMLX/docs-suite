# rightSplit

@short: Optional. Sets the number of frozen columns from the right

@signature: {'rightSplit?: number;'}

@example:
const grid = new dhx.Grid("grid_container", {
    columns: [
        // columns config
    ],
    rightSplit: 2,
    data: dataset
});
