# editable

@short: Optional. Enables editing in Grid columns

@signature: {'editable?: boolean;'}

@default: false

@example:
const grid = new dhx.Grid("grid_container", {
	columns: [
		// columns config
	],
	editable:true,
	data: dataset
});

@descr:
