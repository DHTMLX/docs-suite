# leftSplit

@short: Optional. Sets the number of frozen columns from the left

@signature: {'leftSplit?: number;'}

@example:
const grid = new dhx.Grid("grid_container", {
	columns: [
		// columns config
	],
	leftSplit:2,
	data: dataset
});

@descr:
