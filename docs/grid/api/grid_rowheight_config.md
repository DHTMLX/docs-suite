# rowHeight

@short: defines the height of a row in a grid

@signature: {'rowHeight?: number;'}

@default: 40

@example:
var grid = new dhx.Grid("grid_container", {
	columns: [
		// columns config
	],
	rowHeight: 30,
	data: dataset
});

@descr:
