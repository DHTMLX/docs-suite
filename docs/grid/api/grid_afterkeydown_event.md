# afterKeyDown

@short: fires after the user is pressing a shortcut key

@signature: {'afterKeyDown: (e: Event) => void;'}

@params:
- `e: Event` - a native KeyboardEvent object

@example:
grid.events.on("afterKeyDown", function (e) {
    // your logic here
});
