# isRowHidden()

@short: checks whether a row is hidden

@signature: {'isRowHidden(rowId: string | number): boolean;'}

@params:
- `rowId: string | number` - the id of a row

@returns:
`true`, if the row is hidden; `false` otherwise.

@example:
grid.isRowHidden("1"); //-> true/false
