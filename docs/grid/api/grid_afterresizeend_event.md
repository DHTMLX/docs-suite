# afterResizeEnd

@short: fires after resizing of a column is ended

@signature: {'afterResizeEnd: (col: object, e: MouseEvent) => void;'}

@params:
- `col: object` - an object with a column configuration
- `e: MouseEvent` - a native HTML event object

@example:

	// your logic here
