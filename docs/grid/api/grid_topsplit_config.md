# topSplit

@short: Optional. Sets the number of frozen rows from the top

@signature: {'topSplit?: number;'}

@example:
const grid = new dhx.Grid("grid_container", {
	columns: [
		// columns config
	],
	topSplit: 2,
	data: dataset
});

@descr:
