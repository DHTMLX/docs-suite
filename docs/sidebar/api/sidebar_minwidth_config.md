# minWidth

@short: Optional. Sets the minimal width of a sidebar in the collapsed state

@signature: {'minWidth?: number | string;'}

@default: "44"

@example:
const sidebar = new dhx.Sidebar("sidebar_container", {
    css: "dhx_widget--border_right",
    minWidth: 80,
    collapsed: true
});
