# menuCss

@short: Optional. Adds style classes to all containers of Sidebar controls with nested items 

@signature: {'menuCss?: string;'}

@example:
<style>
    .my_first_class {
        /*some styles*/
    }
 
    .my_second_class {
        /*some styles*/
    }
</style>

const sidebar = new dhx.Sidebar("sidebar_container", {
    menuCss:"my_first_class my_second_class"
});
