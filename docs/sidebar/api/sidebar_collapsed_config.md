# collapsed

@short: defines that a sidebar is initialized in the collapsed state

@signature: {'collapsed?: boolean;'}

@example:
var sidebar = new dhx.Sidebar(document.body, {
    css: "dhx_widget--border_right",
    collapsed: true
});
