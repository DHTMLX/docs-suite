# beforeExpand

@short: fires before expanding a sidebar

@signature: {'beforeExpand: () => boolean | void;'}

@returns:
Return `false` to prevent expanding a sidebar; otherwise, `true`.

@example:
sidebar.events.on("beforeExpand", function() {
    // your logic here
    return true;
});

@descr:
