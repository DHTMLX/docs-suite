# getSelected()

@short: returns an array of IDs of selected items

@signature: {'getSelected(): (string | number)[];'}

@returns:
An array of string/number values with IDs of selected items.

@example:
menu.getSelected(); // -> ["selected_1", "selected_1.1"]
