# beforeOpen

@short: fires before opening a list with options

@signature: {'beforeOpen: () => boolean | void;'}

@returns:
Return `false` to prevent opening a combobox; otherwise, `true`.

@example:
combo.events.on("beforeOpen", function() {
    // your logic here
    return false;
});

@descr:
