# isFullScreen()

@short: checks whether the window is in the full screen mode

@signature: isFullScreen(): boolean;

@returns:
`true`, if a window is in the full screen mode; otherwise, `false`.

@example:
dhxWindow.isFullScreen(); // -> true/false

@descr:
