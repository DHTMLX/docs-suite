# setSize()

@short: sets the size of a window

@signature: {'setSize(width: number, height: number): void;'}

@params:
- `width: number` - the width of a window
- `height: number` - the height of a window

@example:
dhxWindow.setSize(250, 250);
dhxWindow.show();
