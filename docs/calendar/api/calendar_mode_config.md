# mode

@short: Optional. The mode of Calendar initialization

@signature: {'mode?: "calendar" | "month" | "year" | "timepicker";'}

@default: "calendar"

@example:
const calendar = new dhx.Calendar("calendar_container", {
    mode: "month"
});
