# range

@short: Optional. Enables/disables the possibility to select a range of dates on the calendar

@signature: {'range?: boolean;'}

@default: false

@example:
const calendar = new dhx.Calendar("calendar_container", {
    css: "dhx_widget--bordered",
    range: true
});

@descr:
