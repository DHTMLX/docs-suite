# thisMonthOnly

@short: Optional. Hides dates of the previous/next months relative to the currently displayed one

@signature: {'thisMonthOnly?: boolean;'}

@default: false

@example:
const calendar = new dhx.Calendar("calendar_container", {
   thisMonthOnly:true
});

@descr:
