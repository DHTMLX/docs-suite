# addCell()

@short: adds a new cell into a layout

@signature: {'addCell(config: ICellConfig, index?: number): void;'}

@params:
- `config: object` - an object with configuration properties of a cell. Check [the full list of available properties](../../../category/layout-cell-properties/)
- `index: number` - optional, the position to add a cell into



@example:
layout.addCell({
    header: "New layout cell"
});
