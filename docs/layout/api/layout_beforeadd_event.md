# beforeAdd

@short: fires before adding a cell

@signature: {'beforeAdd: (id: string) => boolean | void;'}

@params:
- `id: string` - the id of a cell

@returns:
Return `false` to prevent adding of a cell; otherwise, `true`.

@example:
layout.events.on("beforeAdd", function(id){
	// your logic here
    return false;
});

@descr:
