# headerIcon

@short: Optional. An icon used in the header of a cell

@signature: {'headerIcon?: string;'}

@example:
const layout = new dhx.Layout("layout_container", {
    cols: [
        { header:"Cell header", headerIcon:"dxi dxi-check"}
    ]
});

@descr:
