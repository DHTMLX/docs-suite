# collapsed

@short: Optional. Defines whether a cell is collapsed

@signature: {'collapsed?: boolean;'}

@example:
const layout = new dhx.Layout("layout_container", {
    cols: [
        { header: "Cell header", collapsed:true}
    ]
});
