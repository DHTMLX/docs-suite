# padding

@short: Optional. Defines the distance between a cell and the border of layout

@signature: {'padding?: number | string;'}

@example:
const layout = new dhx.Layout("layout_container",{
	padding: 10,
	css: "dhx_layout-cell--bordered",
	cols: [
		{
            padding: 10,
            css: "dhx_layout-cell--bordered",
            html: "First Block"
        },
        {
            padding: 10,
            css: "dhx_layout-cell--bordered",
            html: "Second Block"
        }
    ]
});

@descr:
