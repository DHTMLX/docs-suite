# resizable

@short: Optional. Defines whether a cell can be resized

@signature: {'resizable?: boolean;'}

@example:
const layout = new dhx.Layout("layout_container", {
    cols: [
        { header: "Cell header", resizable:true}
    ]
});

@descr:
