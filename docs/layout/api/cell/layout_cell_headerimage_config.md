# headerImage

@short: Optional. An image used in the header of a cell

@signature: {'headerImage?: string;'}

@example:
const layout = new dhx.Layout("layout_container", {
    cols: [
        { header: "Cell header", headerImage:"/img.png"}
    ]
});
