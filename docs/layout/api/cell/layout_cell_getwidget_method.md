# getWidget()

@short: returns the widget attached to a layout cell

@signature: {'getWidget(): any;'}

@returns:
An object of a widget attached to a cell.

@example:
// id - the id of a cell a widget is attached to
var widget = layout.getCell("id").getWidget();
