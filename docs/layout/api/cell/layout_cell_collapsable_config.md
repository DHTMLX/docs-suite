# collapsable

@short: Optional. Defines whether a cell can be collapsed

@signature: {'collapsable?: boolean;'}

@example:
const layout = new dhx.Layout("layout_container", {
    cols: [
        { header: "Cell header", collapsable:true}
    ]
});
