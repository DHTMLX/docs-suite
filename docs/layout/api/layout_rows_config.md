# rows

@short: Optional. An array of rows objects

@signature: {'rows?: object[];'}

@example:
const layout = new dhx.Layout("layout_container", {
    header: "Main Block Header",
    rows: [
        {
            cols: [
                {
                    header: "Block 1",
                    width: "40%"                                
                },
                {
                    header: "Block 2",                  
                    width: "60%"
                }
            ]
        },
        {
            rows: [
                {
                    header: "Block 3",
                    height: "200px"                             
                },
                {
                    header: "Block 4",                              
                    height: "300px"
                }
            ]
        }
    ]
});

@descr:
