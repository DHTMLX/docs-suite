# maxPoints

@short: Optional. Displays an average number of values in case a data set is too large to show all the values in the chart

@signature: {'maxPoints?: number;'}

@example:
const chart = new dhx.Chart("chart_container", {
	type:"line",
	maxPoints:100
});

@descr:
