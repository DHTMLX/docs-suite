# type

@short: specifies the type of a chart

@signature: {'type?: ChartType;'}

@example:
var chart = new dhx.Chart("chart_container",{
    type:"line" 
});

@descr:
## Available types of charts 

- "bar", "x-bar"
- "line", "spline"
- "scatter"
- "area", "splineArea"
- "donut", "pie", "pie3D"
- "radar"
