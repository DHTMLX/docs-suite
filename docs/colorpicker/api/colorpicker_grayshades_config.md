# grayShades

@short: defines whether the section with gray shades is displayed in the palette

@signature: {'grayShades?: boolean;'}

@default: true

@example:
const colorpicker = new dhx.Colorpicker("colorpicker_container", {
	grayShades: false
});

@descr:
