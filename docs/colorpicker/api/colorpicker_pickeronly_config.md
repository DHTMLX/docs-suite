# pickerOnly

@short: Optional. Defines whether Colorpicker is shown only in the picker mode

@signature: {'pickerOnly?: boolean;'}

@default: false

@example:
const colorpicker = new dhx.Colorpicker("colorpicker_container",{
	pickerOnly: true
});

@descr:
