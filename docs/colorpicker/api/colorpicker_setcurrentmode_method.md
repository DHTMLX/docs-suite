# setCurrentMode()

@short: shows colopicker either in the "palette" or in the "picker" mode

@signature: {'setCurrentMode(view: "palette" | "picker"): void;'}

@params:
- `view: string` - the mode to show: "palette" or "picker"


@example:
colorpicker.setCurrentMode("picker");
