# transparency

@short: Optional. Defines whether the transparency scale is displayed in the picker mode

@signature: {'transparency?: boolean;'}

@default: true

@example:
const colorpicker = new dhx.Colorpicker("colorpicker_container",{
	transparency: false
});

@descr:
