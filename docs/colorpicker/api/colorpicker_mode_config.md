# mode

@short: specifies the mode of displaying a colorpicker

@signature: {'mode?: "palette" | "picker";'}

@example:
var colorpicker = new dhx.Colorpicker("colorpicker", {
	mode: "palette",
	css: "dhx_widget--bordered"
});


@descr:
