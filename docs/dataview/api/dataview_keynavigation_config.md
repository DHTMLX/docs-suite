# keyNavigation

@short: Optional. Enables/disables navigation in DataView by arrow keys

@signature: {'keyNavigation?: boolean | (() => boolean);'}

@default: true

@example:
const dataview = new dhx.DataView("dataview_container", {
    itemsInRow: 5, 
    keyNavigation: true
});

@descr:
