# keyNavigation

@short: enables/disables navigation in DataView by arrow keys

@signature: {'keyNavigation?: boolean | (() => boolean);'}

@example:
var dataview = new dhx.DataView("dataview", {
    itemsInRow: 5, 
    keyNavigation: true
});

@descr:
