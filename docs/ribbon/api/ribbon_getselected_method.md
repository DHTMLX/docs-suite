# getSelected()

@short: returns an array of IDs of selected items

@signature: {'getSelected(): string[];'}

@returns:
An array of string values with IDs of selected items.

@example:
ribbon.getSelected(); // -> ["selected_1", "selected_1.1"]

@descr:

@changelog:
added in v7.0
