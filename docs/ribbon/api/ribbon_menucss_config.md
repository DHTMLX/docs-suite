# menuCss

@short: Optional. Adds style classes to all containers of Ribbon controls with nested items

@signature: {'menuCss?: string;'}

@example:
<style>
    .my_first_class {
        /*some styles*/
    }
 
    .my_second_class {
        /*some styles*/
    }
</style>

const ribbon = new dhx.Ribbon("ribbon_container", {
    menuCss:"my_first_class my_second_class"
});
